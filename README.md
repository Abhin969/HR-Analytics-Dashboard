# 📊 Recruitment, Performance & HR Insights Dashboard

## 📌 Overview
This Power BI dashboard provides insights into employee recruitment, performance, salary distribution, and HR analytics.

## 🎯 Key Features
- Employee Count & Salary Overview
- Recruitment Source Analysis
- Gender Distribution
- Absence Analysis by Citizenship
- Department-wise Performance Insights
- Marital Status Distribution

## 📈 KPIs
- Total Employees: 311
- Average Salary: 69.02K
- Total Salary: 21M
- Total Leave Days: 3184
